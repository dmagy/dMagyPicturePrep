# dMPMS Draft Archives

These files are historical draft specifications only.

The current public dMPMS standard is:

```text
Docs/dMPMS/dMPMS-v1.0.md
```

Archived draft version numbers such as `1.1`, `1.2`, `1.3`, or `1.4` must not be used as current implementation guidance.

Use these drafts only for historical migration research, archaeology, or understanding how older internal sidecars may have evolved before the public v1.0 standard.

Current implementation guardrail:

- dMPP currently writes `dmpmsVersion: "1.0"`.
- Older draft specs do not change that.
- Any future public dMPMS version bump should be an explicit new design decision, not an inference from these archived files.
