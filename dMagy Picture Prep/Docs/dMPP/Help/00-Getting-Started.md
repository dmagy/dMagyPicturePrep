# Getting Started

Welcome to **dMagy Picture Prep**. dMPP helps you prepare a photo archive so it is easier to understand, search, organize, and display later.

You can use dMPP to add titles, descriptions, dates, people, locations, tags, and crops to your pictures. dMPP saves this information beside each picture in a readable sidecar file. Your original picture files are not changed.

## The basic workflow

1. Choose your **Picture Library Folder**.
2. Add a few key **People**.
3. Add one or two saved **Locations**.
4. Choose a working folder of pictures.
5. Review each picture.
6. Click **Next Picture** to save and continue.

You do not need to set up everything before you begin. Start with enough People and Locations to make your first review session useful, then add more as you go.

## Choose your Picture Library Folder

Your Picture Library Folder is the top-level folder that contains the pictures you want to prepare.

Good examples:

- `Family Photos`
- `Photo Archive`
- `Scanned Pictures`
- `Pictures to Organize`

Less helpful examples:

- one small event folder, such as `Christmas 1998`
- a temporary Downloads folder
- a folder you plan to move or delete soon

dMPP creates or uses a folder named:

```text
dMagy Portable Archive Data
```

inside your Picture Library Folder. This folder stores shared People, Locations, Tags, Crops, and other archive data.

## If your pictures are in Apple Photos

dMPP does not work directly inside the Apple Photos library.

If your pictures are in Photos, export them to regular folders first. Then choose the folder structure that contains those exported pictures as your Picture Library Folder.

This matters because dMPP saves sidecar files next to your pictures. Apple Photos does not work like a normal folder of image files.

## Add key People

Start with a few people who appear often in your pictures.

Good first People to add:

- yourself
- close family members
- frequent friends
- people who appear in many photos

You do not need to build a complete family tree first. Add people as you need them.

People are shared across your whole Picture Library Folder, so you create each person once and reuse them wherever they appear.

## Add saved Locations

Add one or two places that appear often in your pictures.

Examples:

- Home
- Grandma’s House
- Church
- School
- Lake Cabin

Saved Locations help keep location names consistent. They also make GPS-derived locations more meaningful when a picture contains GPS metadata.

## Start reviewing pictures

After setup, choose a working folder inside your Picture Library Folder.

A working folder is the folder of pictures you want to review right now. For a first session, choose a small folder with 10–20 pictures. That gives you room to learn the workflow without feeling buried.

For each picture, review what you know:

- Title
- Description
- Date Taken or Era
- Tags
- People
- Location
- Crops
- Curator Notes, if needed

You do not need to fill in every field for every picture.

## Saving

The usual workflow is simple:

```text
Review picture → Click Next Picture → dMPP saves and moves forward
```

You can also click **Save** manually when you want to write changes before closing the app, switching folders, or doing something else.

dMPP saves your work in a `.dmpms.json` sidecar file beside the picture. The original image is not changed.

## What to learn later

You can start reviewing pictures before mastering every feature.

Learn these as they become useful:

- Suggested vs Manual People modes
- Tags
- Virtual crops
- Headshot crops
- Curator Notes
- Folder access troubleshooting
- Advanced Settings

Use the section help buttons in the editor when a specific question comes up.
